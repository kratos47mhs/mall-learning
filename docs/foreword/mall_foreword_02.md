# mall learning knowledge points

> Since the mall project involves many knowledge points, such as Spring Boot, Elastic Search, Redis, Mongodb, etc., this tutorial will not describe these in detail, but only the relevant parts of this project, so the following information is recommended. For those who are not familiar with some of these knowledge points, you can look at the recommended materials below.

## Recommended materials

### IDEA
《IntelliJ-IDEA-Tutorial》：[https://www.youtube.com/user/intellijideavideo/playlists](https://www.youtube.com/user/intellijideavideo/playlists)
> Especially complete IDEA tutorials, you can learn many practical skills.

### Spring
《Spring in Action (4th edition)》：[https://www.amazon.com/dp/1933988134](https://www.amazon.com/dp/1933988134)
> Classic, best-selling Spring learning and practice guide, from this book you can learn the practical usage of Spring, have an overall understanding of Spring, recommend reading the entire book.

### SpringBoot
《Spring Boot in Action》：[https://www.amazon.com/dp/1617292540/](https://www.amazon.com/dp/1617292540/)
> Spring Boot's introductory book has a total of more than 200 pages. Anyway, I have read it, and the part about Groovy and Grails is not necessary.

### MyBatis
《Java Persistence with MyBatis 3》：[https://www.amazon.com/Java-Persistence-MyBatis-Prasad-Reddy-ebook/dp/B00DIY89P8/](https://www.amazon.com/Java-Persistence-MyBatis-Prasad-Reddy-ebook/dp/B00DIY89P8/)
> Good coverage of annotations and MyBatis (something that is sorely lacking in the documentation).

### MySql
《Learning MySQL: Get a Handle on Your Data》：[https://www.amazon.com/Learning-MySQL-Handle-Your-Data-ebook/dp/B004LRPBCK/](https://www.amazon.com/Learning-MySQL-Handle-Your-Data-ebook/dp/B004LRPBCK/)
> A MySql book. As a developer, we only need to look at the first part of the basics, the second part of the development, and the third part of the optimization.

### Linux
《The Linux Command Line, 2nd Edition》：[https://www.amazon.com/Linux-Command-Line-2nd-Introduction/dp/1593279523/](https://www.amazon.com/Linux-Command-Line-2nd-Introduction/dp/1593279523/)
> This was an excellent introduction to the Linux command line and bash scripting.

### Elasticsearch
《Elasticsearch Reference》：[https://www.elastic.co/guide/en/elasticsearch/reference/current/index.html](https://www.elastic.co/guide/en/elasticsearch/reference/current/index.html)
> The learning materials officially recommended by Elasticsearch are based on Elasticsearch 2.4.x version, which is relatively old but can be used. For most mature frameworks, the version iteration usage will not differ much.

《Elasticsearch in Action》：[https://www.amazon.com/Elasticsearch-Action-Radu-Gheorghe/dp/1617291625/](https://www.amazon.com/Elasticsearch-Action-Radu-Gheorghe/dp/1617291625/)
> If you think the above version of Elastic Search is too old, you can read this one.

### Mongodb
《MongoDB in Action: Covers MongoDB version 3.0》：[https://www.amazon.com/MongoDB-Action-Covers-version-3-0/dp/1617291609/](https://www.amazon.com/MongoDB-Action-Covers-version-3-0/dp/1617291609/)
> A very good Mongo DB combat book, the author has participated in Mongo DB driver development, you can read all those interested.

### Docker
《Hands-On Microservices with Spring Boot and Spring Cloud: Build and deploy Java microservices using Spring Cloud, Istio, and Kubernetes》：[https://www.amazon.com/Hands-Microservices-Spring-Boot-Cloud-ebook/dp/B07T1Y2JRJ](https://www.amazon.com/Hands-Microservices-Spring-Boot-Cloud-ebook/dp/B07T1Y2JRJ)
> We only need to look at the Docker part of this book, which will be used later in the production environment deployment project.

## Conclusion
> If you read the information in the above part according to my recommendation, or you already have the foundation of the above part, then you will be very smooth when learning mall.
